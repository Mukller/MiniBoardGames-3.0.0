🔑 СПОСОБ 1: PUSH ЧЕРЕЗ HTTPS + GitHub Token
════════════════════════════════════════════════════════════════════════════════

Требуется:
- GitHub Personal Access Token (классическое или Fine-grained)
- Или GitHub пароль (если включён)

Шаги:
1. Создать GitHub Personal Access Token:
   - Откройте: https://github.com/settings/tokens
   - Нажмите: "Generate new token" → "Generate new token (classic)"
   - Разрешите: repo (полный доступ к репозиториям)
   - Скопируйте токен

2. Выполните пуш:
   cd /home/claude/MiniBoardGames-3.0.0-git
   git push origin main

3. При запросе пароля:
   - Username: Mukller
   - Password: [Вставьте скопированный токен]

════════════════════════════════════════════════════════════════════════════════

🔑 СПОСОБ 2: PUSH ЧЕРЕЗ SSH (более безопасно)
════════════════════════════════════════════════════════════════════════════════

Требуется:
- SSH ключ добавлен в GitHub
- SSH агент запущен

Шаги:
1. Если SSH ключ ещё не добавлен в GitHub:
   - Откройте: https://github.com/settings/keys
   - Нажмите: "New SSH key"
   - Вставьте содержимое ~/.ssh/id_rsa.pub
   - Сохраните

2. Измените URL на SSH:
   cd /home/claude/MiniBoardGames-3.0.0-git
   git remote set-url origin git@github.com:Mukller/MiniBoardGames-3.0.0.git

3. Проверьте URL:
   git remote -v

4. Пушьте изменения:
   git push origin main
