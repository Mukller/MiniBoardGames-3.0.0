# MiniBoardGames 3.0.0

Плагин для добавления мини-игр прямо в Minecraft сервер!

## 📋 Описание

MiniBoardGames - это мощный плагин для Minecraft серверов, который позволяет игрокам участвовать в различных увлекательных мини-играх. Идеален для развлечения игроков и создания активного сообщества на вашем сервере.

## ✨ Основные возможности

- 🎮 Множество различных мини-игр
- 👥 Поддержка мультиплеера
- ⚙️ Простая конфигурация
- 💾 Сохранение статистики игроков
- 🎯 Система награждения
- 🌍 Поддержка локализации

## 📋 Требования

- **Minecraft Server**: 1.16.5 и выше
- **Java**: Java 11 или выше
- **Bukkit/Spigot/Paper**: Совместимо с популярными фorks

## 🚀 Установка

1. Скачайте последнюю версию плагина
2. Поместите файл `.jar` в папку `/plugins` вашего сервера
3. Перезагрузите сервер командой `/reload` или полностью перезапустите его
4. Плагин автоматически создаст конфигурационные файлы

## 🎮 Основные команды

```
/mbg help              - Справка по плагину
/mbg play <game>       - Начать мини-игру
/mbg list              - Список доступных игр
/mbg stats             - Ваша статистика
/mbg reload            - Перезагрузить конфиг (администратор)
```

## ⚙️ Конфигурация

Основной файл конфигурации находится в `plugins/MiniBoardGames/config.yml`:

```yaml
# Основные настройки
plugin:
  debug: false
  language: ru_RU

# Настройки игр
games:
  enabled-games:
    - jump
    - race
    - puzzle
    - trivia
  
# Награды
rewards:
  enabled: true
  money-per-win: 100
```
