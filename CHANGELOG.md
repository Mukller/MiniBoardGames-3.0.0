## [3.0.0] - 2026-05-14

### Добавлено
- 🎮 Полная переработка системы игр
- 👥 Новая система управления плеями
- 📊 Улучшенная база данных статистики
- ⚙️ Гибкая система конфигурации для каждой игры
- 🏆 Встроенная система наград и достижений
- 🌍 Поддержка локализации (русский, английский)
- 🔧 Горячая перезагрузка конфигураций
- 📈 Детальная статистика игроков

### Изменено
- Переносится с MySQL на SQLite по умолчанию
- Новая архитектура плагина для лучшей производительности
- Улучшены команды игроков

### Исправлено
- Исправлены проблемы с синхронизацией данных
- Исправлены утечки памяти в игровом движке
- Исправлены ошибки при выходе из игры

### Удалено
- Удалена устаревшая система очков
- Удалены неработающие игры

## Легенда

- **Добавлено (Added)** - новая функциональность
- **Изменено (Changed)** - изменения существующей функциональности
- **Исправлено (Fixed)** - исправления ошибок
- **Удалено (Removed)** - удалённая функциональность
- **Устарело (Deprecated)** - функциональность, которая будет удалена в будущем
- **Безопасность (Security)** - исправления уязвимостей

---

## Версионирование

Этот проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html):

- **MAJOR** (3.0.0) - несовместимые изменения API
- **MINOR** (3.1.0) - новая функциональность (совместимая)
- **PATCH** (3.0.1) - исправления ошибок
