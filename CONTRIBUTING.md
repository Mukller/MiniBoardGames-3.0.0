# Руководство по контрибьютингу

Спасибо за интерес к разработке DailyStreak! Мы рады видеть ваш вклад.

## Как начать

1. Форкните репозиторий
2. Создайте ветку для вашей фичи
3. Внесите изменения
4. Отправьте Pull Request

## Требования к коду

- Следуйте стилю Google Java
- Добавляйте комментарии к сложным функциям
- Пишите понятные сообщения в коммитах
- Протестируйте ваши изменения

## Процесс Pull Request

1. Убедитесь, что код компилируется
2. Добавьте описание вашего PR
3. Дождитесь review от мейнтейнера
4. Обсудите предложения улучшений

Спасибо за ваш вклад!
